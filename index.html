CANADIAN COMPUTING COMPETITION

Every year, students of all programming levels are given to opportunity to compete in the Canadian Computing Competition (CCC) which is hosted by the University of Waterloo. This three hour competition consist of five problems students must solve by developing a computational solution. The difficulties ranges from applying easy computation concepts to more challenging algorithms and dynamic programming techniques. The student(s) who score the highest in each provincial groups are awarded a plaque, and students who score above a certain range will also receive special acknowledgements.

Hamber students have demonstrated top performance in the CCC for a total of six times, and are currently on a four year consecutive streak. Majority of the students who won have achieved perfect scores in the competition.

COURSE OUTLINE

Hamber's Department of Computer Science offers three courses. All of these courses are taught in the Linux Lab (Room 307).
The programming languages and libraries used in class are all cross platform, so students are able to
develop and run their applications at home on different Operating Systems (Windows, Mac, Linux).

JUNIOR PROGRAMMING

COURSE CODE: INT10/ICTP11A

This course is designed for students who havenâ€™t had any previous experience in programming, although anyone who wants a review are more than welcomed to joined. Students taking this course will be introduced to fundamental concepts of programming in the Python language, along with learning how to utilize the basics of a UNIX system. By the end of the course students should have a good understanding of the following concepts:

  • Basic UNIX Commands
  • Python Language Fundamentals
  • SSH and remote login
  • Binary Arithmetic
  • TCP and UDP Network

INTERMEDIATE PROGRAMMING

COURSE CODE: ICTP11B / ICTP12A

This is the next level course for programming and is intended for students who have successfully completed the junior course, or students that have demonstrated understanding of junior-level programming concepts. This year students will be introduced to the Object Oriented Programming paradigm and will be able to demonstrate their understanding by utilizing popular libraries such as SFML, Pygame, and FLTK. The purpose of the intermediate course is to have students learn how to develop full applications.

  • Object Oriented Programming concepts
  • Deeper understanding of networking
  • SFML Toolkit
  • Pygame
  • FLTK library

SENIOR PROGRAMMING

COURSE CODE: ICTP12B

This is the final course offered for programming. Students who are taking this course will be introduced to the C/C++ language. Previously used libraries such as FLTK and SFML will be revisited in C++. This course is focused on teaching higher level programming concepts such as dynamic memory allocation, pointers, inheritance, and polymorphism. Students will also learn special skills such as Linux system administration, along with micro computer programming via an Arduino.

  • C/C++ Language Fundamentals
  • More Algorithms
  • Linux System Administration
  • Micro computer programming via Arduino
