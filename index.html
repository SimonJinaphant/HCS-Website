WHAT IS COMPUTER SCIENCE?

Computer science is the study of how computers work, and how we can use them to solve problems.

Computer scientists use a mix of math, logic, and problem-solving strategies
in order to write sequential instructions that computers can follow.

WHY TAKE COMPUTER SCIENCE?

If you plan on going to post-secondary education to study one of the following:

  • Mathematics
  • Engineering
  • Science
  • Commerce
then you will need to take computer science courses. Taking a computer science course in high school can help you get a head start and a solid foundation.

Regardless of your future career choice, everyone can benefit from learning how to code. Computers are used everywhere in society, and understanding how to program computers is a great skill. CANADIAN COMPUTING COMPETITION

Every year, the top computer science students in Canada compete in the prestigious Canadian Computing Competition, hosted by the University of Waterloo. Students work to solve five problems by writing computer programs. The problems vary in complexity, and the top students in each national region are awarded a plaque and prize money.

As of June 2013, Hamber has won the Western Canadian Junior Championship six times. In the 2013 competition, Hamber broke a national record with four perfect scores.

JUNIOR PROGRAMMING

COURSE CODE: INT10/ICTP11A

Junior Programming is designed for students without any previous experience in computer science. Students will be introduced to fundamental concepts of programming using the wonderful Python language. They will also learn the basics of how to use a Linux operating system. Basic hardware concepts such as logic gates and the binary number system will also be introduced. This is a very rich course. Be prepared to learn something new every class. By the end of the course students should have a good understanding of the following concepts:

  • Basic UNIX Commands
  • Python Language Fundamentals
  • Usage of SSH and remote logins
  • Binary Arithmetic
  • TCP and UDP Network Protocols

INTERMEDIATE PROGRAMMING

COURSE CODE: ICTP11B / ICTP12A

Intermediate Programming is intended for students who have successfully completed the junior course. Students will learn how to write GUI (Graphical User Interface) programs using the pyFLTK library. Students will also learn to create arcade style video games using the pygame library. The paradigm of object-oriented programming, or OOP will be introduced. Network programming will also be introduced in this course. The intermediate course teaches students how to develop full applications that make use of available Python libraries. Students should have experience in the following by the end of the year:

  • Object Oriented Programming concepts
  • Utilizing and Managing Virtual Machines
  • Libraries such as SFML, Pygame and FLTK
  • Deeper understanding of networking

SENIOR PROGRAMMING

COURSE CODE: ICTP12B

The final course offered is for experienced students interested in computer science. This course will introduce students to the C/C++ language. Complex topics such as pointers, dynamic memory, classes, inheritance and polymorphism will be taught. In addition, students will learn how to create video games in C++ using the SFML library. This course contains higher level Mathematics such as calculus and matrix algebra. By the end of the year students should have experience in the following:

  • C/C++ Language Fundamentals
  • Dynamic Memory Allocation
  • Advance Algorithms
  • Exposure to post-secondary level Mathematics
